# ATLAS Knowledge Base

Memória de longo prazo do projeto ATLAS — decisões, bugs conhecidos, lições e contexto.

## Regra de entrada
Só entra o que é **não óbvio**, **não está no código**, ou **custou tempo para descobrir**.

## Estrutura
```
architecture/    → Decisões de arquitetura e fluxos de dados
debugging/       → Bugs conhecidos, workarounds, lições
decisions/       → Architecture Decision Records (ADRs)
context/         → Estado atual, ambiente, como rodar
```
