# Implementação do Módulo de Banco de Dados - connection.py

Vou implementar o módulo de conexão com o banco de dados SQLite conforme especificado na v1. Este módulo será responsável por:

1. Criar conexões com os quatro bancos de dados SQLite
2. Configurar as conexões com as opções apropriadas (WAL mode, foreign keys)
3. Garantir que os arquivos de banco de dados sejam criados no diretório correto

A implementação seguirá exatamente o que foi especificado no documento v1, na seção 5 (MÓDULO db/connection.py).
